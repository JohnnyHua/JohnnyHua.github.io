<h3>PCP (Progress Control Plane)</h3>

<div data-lang="en">
<p><strong>Status</strong>: 🔵 In Progress</p>
<p>Protocol layer and state kernel for AI agent collaboration.</p>
<p><strong>Problem it solves</strong>:</p>
<ul>
  <li>AI sessions drift from scope</li>
  <li>Work interrupted cannot resume</li>
  <li>Completion claimed without evidence</li>
  <li>Multi-session development lacks state</li>
</ul>
<p><strong>Core value</strong>:</p>
<ul>
  <li>Single source of truth for task state</li>
  <li>Evidence-first completion rules</li>
  <li>Tool-agnostic protocol</li>
</ul>
<p><strong>Location</strong>: <code>/clawd/pcp/</code></p>
</div>
<div data-lang="zh" style="display: none;">
<p><strong>状态</strong>: 🔵 进行中</p>
<p>AI Agent 协作的协议层和状态内核。</p>
<p><strong>解决的问题</strong>：</p>
<ul>
  <li>AI 会话偏离范围</li>
  <li>工作中断无法恢复</li>
  <li>没有证据就声称完成</li>
  <li>多会话开发缺乏状态</li>
</ul>
<p><strong>核心价值</strong>：</p>
<ul>
  <li>任务状态的单一数据源</li>
  <li>证据优先的完成规则</li>
  <li>工具无关的协议</li>
</ul>
<p><strong>位置</strong>: <code>/clawd/pcp/</code></p>
</div>

<h2 data-lang="en">Why Track Paused Projects?</h2>
<h2 data-lang="zh" style="display: none;">为什么要记录暂停的项目？</h2>

<div data-lang="en">
<p>Because they still have value:</p>
<ul>
  <li>Document what was learned</li>
  <li>Explain why it stopped</li>
  <li>Help future decisions</li>
  <li>Maybe someone else can continue</li>
</ul>
</div>
<div data-lang="zh" style="display: none;">
<p>因为它们仍然有价值：</p>
<ul>
  <li>记录学到了什么</li>
  <li>解释为什么停止</li>
  <li>帮助未来决策</li>
  <li>也许别人可以继续</li>
</ul>
</div>
