<h3>PCP (Progress Control Plane)</h3>

<div data-lang="en">
<p>Protocol layer and state kernel for PM-Agent and Dev-Agent collaboration.</p>
<p><strong>Problem it solves</strong>:</p>
<ul>
  <li>AI sessions drift from scope</li>
  <li>Work gets interrupted and cannot resume</li>
  <li>Completion claimed without verifiable evidence</li>
  <li>Multi-session development lacks consistent state</li>
</ul>
<p><strong>Core value</strong>:</p>
<ul>
  <li>Single source of truth for task state</li>
  <li>Evidence-first completion rules</li>
  <li>Predictable handoff across sessions/models</li>
  <li>Tool-agnostic protocol</li>
</ul>
<p><strong>Status</strong>: In development</p>
</div>
<div data-lang="zh" style="display: none;">
<p>PM-Agent 和 Dev-Agent 协作的协议层和状态内核。</p>
<p><strong>解决的问题</strong>：</p>
<ul>
  <li>AI 会话偏离范围</li>
  <li>工作中断无法恢复</li>
  <li>没有证据就声称完成</li>
  <li>多会话开发缺乏一致状态</li>
</ul>
<p><strong>核心价值</strong>：</p>
<ul>
  <li>任务状态的单一数据源</li>
  <li>证据优先的完成规则</li>
  <li>跨会话/模型的可预测交接</li>
  <li>工具无关的协议</li>
</ul>
<p><strong>状态</strong>：开发中</p>
</div>
